## ğŸ¤ Contributing

We welcome contributions! Here's how you can help:

### Adding Resources

1. **Videos**: Add to `_data/videos.yml`
2. **Standards**: Add to `_data/standards.yml`
3. **Research Papers**: Add to `_data/research.yml`
4. **Glossary Terms**: Add to `glossary.md`

### Quality Guidelines

- **Authoritative Sources**: Only include resources from reputable organizations, academic institutions, or peer-reviewed publications
- **Relevance**: Ensure resources align with the academic frameworks
- **Accuracy**: Verify information before adding
- **Accessibility**: Prefer open-access resources when possible
- **Currency**: Prioritize recent publications (last 5 years)

### Resource Format

When adding resources, include:
- **Title**: Full title of the resource
- **URL**: Direct link
- **Source**: Organization or publisher
- **Date**: Publication date
- **Description**: Brief summary (1-2 sentences)
- **Category**: Which framework category it belongs to
